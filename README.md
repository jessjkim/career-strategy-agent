# Resume Strategy Agent

Resume Strategy Agent helps job seekers turn a resume into a focused search plan. This app is powered by an AI agent (Agno + OpenAI). The agent reads the resume, extracts structured signals (role, skills, strengths), synthesizes industry trends from news headlines, and generates a targeted company radar with actionable recommendations.

Agno provides the agent orchestration layer. The agent receives the resume text and the user’s target preferences, then outputs JSON for strengths, strategy, keywords, and company targets. A second agent pass summarizes industry news into a short ‘pulse’ report. All outputs are generated by the model and rendered directly in the UI.

Use it to:
- Clarify what roles you’re best positioned for
- Build a targeted company list instead of spray‑and‑pray
- Stay current on industry trends that shape hiring

## Features
- PDF resume upload (in‑memory).
- Auto‑suggested target role from resume.
- Industry + location multi‑selects.
- Resume strengths + strategy tied to industry trends.
- Industry news summary + top links.
- Target company list derived from resume + industry discovery.

## Typical workflow
1) Upload your resume.  
2) Confirm or adjust the suggested target role.  
3) Pick industries and locations.  
4) Run analysis to get strengths, strategy, industry pulse, and company targets.
